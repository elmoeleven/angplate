# Angplate

A wrapper that gives you a base AngularJS/Rails/PIXI.js app.

## Setup
```bash
# Install npm
$ brew install npm

# Install bower
$ npm install bower -g

# Clone it
$ g clone https://github.com/elmoeleven/angplate.git

# Go in
$ cd angplate

# Rails setup
$ rake db:create; rake db:migrate
$ bundle install

# Dependency setup
$ rake bower:install

# Start your server!
$ rails s
```
